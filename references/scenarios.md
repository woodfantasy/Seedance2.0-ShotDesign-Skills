# 垂直场景提示词模板库

> 针对高频使用场景提供开箱即用的提示词架构参考。每个模板包含核心参数和示例。

---

## 一、电商产品广告

**核心参数：**
- 画幅：9:16竖屏（社交投放）或 16:9横屏（官网/TVC）
- 时长：5-10秒
- 运镜：Orbit环绕 + 微距特写 + Hero Lighting
- 材质：注重产品表面质感（金属反光/玻璃折射/液体流动）

**模板：**
```
[时长][产品类目]广告，
Octane物理渲染引擎，hero lighting产品打光，
0-3秒：[产品]居中旋转展示，Orbit 360°环绕，
纯黑/纯白背景，rim light勾勒产品轮廓，
微弱大气粒子漂浮；
4-7秒：Extreme Close-Up微距特写，
[产品表面材质细节]，浅景深散景，
光泽/纹理清晰可见；
8-10秒：Pull Out全景揭示产品在[使用场景]中的效果。
```

**示例 — 腕表广告：**
```
10秒高端腕表广告，Octane物理渲染引擎，hero lighting，
0-3秒：腕表居中Orbit慢速360°环绕，纯黑背景，
rim light精准勾勒表壳金属边缘，微弱金色粒子漂浮；
4-7秒：ECU微距特写表盘，浅景深聚焦秒针，
蓝宝石镜面折射光斑，齿轮机芯细节可见；
8-10秒：Dolly Out，腕表佩戴在手腕上，
手轻抬于窗边golden hour侧逆光中，皮肤纹理自然。
```

## 三、短剧/对白场景

**核心参数：**
- 画幅：9:16竖屏
- 时长：8-15秒
- 运镜：正反打 + Medium Shot + 手持微晃
- 关键：台词用引号包裹并标注角色和情绪

**模板：**
```
[时长]短剧片段，竖屏9:16，[风格]，
画面（0-Xs）：[景别]+[角色A描述]+[场景]，[运镜]
台词1（角色A，[情绪]）："[台词内容]"
画面（X-Xs）：[景别切换]+[角色B反应]+[运镜]
台词2（角色B，[情绪]）："[台词内容]"
音效：[环境音描述]
```

**示例 — 反转短剧：**
```
12秒都市短剧，竖屏9:16，写实自然光风格，
画面（0-4秒）：MS中景，豪华办公室，穿西装的男人背对镜头，
Slow Push In缓慢推进。
台词1（男人，冷漠）："从今天起，你不用来上班了。"
画面（5-8秒）：OTS过肩，对面女生惊愕的表情特写，
手持微晃，浅景深。
台词2（女生，颤抖）："为什么？"
画面（9-12秒）：MS，男人转身，递出一份文件，微笑，
Dolly Out拉出。
台词3（男人，温柔）："因为这是你的CEO任命书。"
```

## 十、超长视频分段拼接模板（>15秒）

**核心策略：** 每段 ≤15秒，使用"视频延长"功能串接。

**Segment 1（主体生成）：**
```
[15秒][完整提示词]，
最后2秒：画面稳定过渡至可延续的交接帧，
角色动作收束至静止/可持续状态，光照稳定。
```

**Segment 2+（延长续接）：**
```
将@视频1延长[X]秒，
保持一致的角色外貌/服装/光照条件/色调风格，
[续接动作描述]，[运镜承接]，
全程延续上段的大气效果和色调。
```

**交接帧原则：**
1. 前段末尾画面必须稳定（禁止运镜中断）
2. 后段开头声明一致性（角色/光照/氛围）
3. 运镜连续（前段Dolly In结束 → 后段自然承接）
